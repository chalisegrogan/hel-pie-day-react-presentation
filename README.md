## Getting Started

1. Clone this repo

   ```sh
   git clone https://github.com/chalisegrogan/hel-pie-day-react-presentation.git
   ```

2. Install dependencies

   ```sh
   yarn install
   ```

3. Start the webpack server. The server will run at [`localhost:3000`](http://localhost:3000).

   ```sh
   yarn start
   ```
